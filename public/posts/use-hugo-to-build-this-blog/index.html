使用 Hugo 來建立這個部落格網站

2023-01-21

目標

  • 文章可以用 markdown 編輯
  • 文章可以根據 tag 做分類
  • 串接 Google Analytics
  • 擁有自己的網址

先決條件

  • 作業系統：macOS
  • 下載 Hugo: brew install hugo

建立部落格

hugo new site blog
cd blog
git init
git submodule add  https://github.com/monkeyWzr/hugo-theme-cactus.git themes/cactus
echo "theme = 'cactus'" >> config.toml
hugo server

你可以在Themes選擇一個你看起來順眼的主題來使用 我是選擇看起來比較簡單的cactus

許多設定都可以在 config.toml 裡調整
